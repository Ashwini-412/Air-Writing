# Air-Writing

A browser-based AR finger-writing application built with MediaPipe Hands.

## Features
- Real-time hand tracking using MediaPipe.
- Smooth drawing with neon glow effects.
- Interactive UX: Hold **Shift** to draw, press **Space** to clear.

## How to use
1. Open `writing.html` in a web browser.
2. Allow camera access.
3. Hold the **Shift** key and move your index finger to write in the air.
4. Press **Space** to clear the canvas.

## Tech Stack
- HTML5 / CSS3
- JavaScript (Vanilla)
- [MediaPipe Hands](https://google.github.io/mediapipe/solutions/hands)
